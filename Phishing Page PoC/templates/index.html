What is Browser Autofill?

Browser autofill is a convenience feature that automatically fills in forms with saved information like usernames, passwords, addresses, and credit card details. While convenient, it poses security risks when exploited by malicious actors.

The Vulnerability

Attackers can create fake login pages that mimic legitimate websites. When users visit these pages, their browsers automatically fill in saved credentials. The fake page can then capture this information without the user actively submitting the form.
